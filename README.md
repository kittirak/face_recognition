# Face Recognition via Deepface

1. clone this repository
   - `git clone https://github.com/kittirak/face_recognition`
2. Prepare "dataset", ถ่ายรูปใบหน้า แล้วเก็บในไดเรอทอรีย่อยภายใต้ไดเรกทอรี dataset ตั้งชื่อ sub directory ตามชื่อเจ้าของใบหน้า 
3. ติดตั้ง deepface `pip install deepface`
4. ทดลองรันไฟล์ run.py และ timeattendance.py
